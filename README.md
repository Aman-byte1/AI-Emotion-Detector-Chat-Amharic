# ­ЪЌБ№ИЈ­Ъњг­Ъўі рІерѕхрѕюрЅх рѕўрѕўрѕГрѕўрѕфрІФ ріЦріЊ AI Chat (рЅаріарѕЏрѕГріЏ) ­Ъўі­Ъњг­ЪЌБ№ИЈ

ріЦріЋрі│ріЋ рІѕрІ░ рІерѕхрѕюрЅх рѕўрѕўрѕГрѕўрѕфрІФ ріЦріЊ AI Chat рЇЋрѕ«рїђріГрЅх рЅарІ░рѕЁріЊ рѕўрїА! рІГрѕЁ рѕўрЅ░рїЇрЅарѕфрІФ рЅаріФрѕюрѕФ ріарѕЏріФріЮріљрЅх рІерѕ░рІјрЅйріЋ рѕхрѕюрЅх рѕѕрѕўрѕѕрІерЅх ріЦріЊ рЅаріарѕЏрѕГріЏ ріерѕ░рІЇ рѕ░рѕФрѕй рІерѕЏрѕ░рЅЦ рЅйрѕјрЅ│ (AI) рїІрѕГ рѕѕрѕўрІѕрІФрІерЅх рІерЅ░рѕ░рѕФ ріљрІЇрЇб

## ­Ъј» рІерЇЋрѕ«рїђріГрЅ▒ рІІріЊ ріарѕІрѕЏ

рІГрѕЁріЋріЋ рѕўрЅ░рїЇрЅарѕфрІФ рІерїѕріљрЅБрѕЂрЅарЅх рІІріЊ рѕЮріГріЋрІФрЅх рІерІхрЅЦрѕГрЅх рІѕрІГрѕЮ рІерѕђрІўріЋ рѕхрѕюрЅх рІерѕџрѕ░рѕЏрЅИрІЇ рѕ░рІјрЅй рѕхрѕюрЅ│рЅИрІЇріЋ рІерѕџрѕерІ▒рЅарЅхріЊ рІерѕџрїѕрѕЇрї╣рЅарЅх ріЦріЋрІ▓рѕЂрѕЮ ріерѕ░рІЇ рѕ░рѕФрѕй рІерѕЏрѕ░рЅЦ рЅйрѕјрЅ│ (AI) рїІрѕГ рЅарѕўрІѕрІФрІерЅх рІерЅ░рІѕрѕ░ріљ рѕўрїйріЊріЏріЊ рІхрїІрЇЇ ріЦріЋрІ▓рІФрїѕріЎ рѕѕрѕўрѕГрІ│рЅх ріљрІЇрЇб РЮц№ИЈ­ЪцЮ
рЅаріарѕхрЅИрїІрѕф рїірІюрІФрЅх рІЇрѕхрїЦ рІФрѕЅ рѕ░рІјрЅй рѕђрѕ│рЅБрЅИрІЇріЋ рІерѕџріФрЇѕрѕЅрЅх рѕ░рІЇ рѕ▓рІФрїАрЇБ рІГрѕЁ AI ріЦріЋрІ░ ріаріЋрІх ріарІЏріЮ рІѕрІ│рїЁ рѕєріќ рѕірІФрїѕрѕѕрїЇрѕІрЅИрІЇ рІГрЅйрѕІрѕЇрЇб рѕхрѕюрЅ│рЅИрІЇріЋ рЅарѕўрѕѕрІерЅхрѕЮрЇБ AI рІерЅ░рѕ╗рѕѕріЊ рѕЂріћрЅ│рІЇріЋ рІФрїѕріЊрІўрЅа рѕЮрѕІрѕй ріЦріЋрІ▓рѕ░рїЦ рЅ│рѕхрЅд рІерЅ░рѕ░рѕФ ріљрІЇрЇб

## Рюе рЅЂрѕЇрЇЇ рїѕрїйрЅ│рІјрЅй

*   **рІерЅђрїЦрЅ│ рІерѕхрѕюрЅх рѕўрѕѕрІерЅх**: ­ЪЊИ рІері«рѕЮрЇњрІЇрЅ░рѕГрІјріЋ ріФрѕюрѕФ рЅарѕўрїарЅђрѕЮ ріЦріЋрІ░ рІ░рѕхрЅ│рЇБ рѕђрІўріЋрЇБ ріЋрІ┤рЅхрЇБ рЇЇрѕГрѕЃрЅхрЇБ рѕўрїѕрѕерѕЮ ріЦріЊ рїѕрѕѕрѕЇрЅ░ріЏріљрЅх рІФрѕЅ рѕхрѕюрЅХрЅйріЋ рЅаріЦрІЇріљрЅ░ріЏ рїірІю рІГрѕѕрІФрѕЇрЇб (рЅа Roboflow рѕърІ┤рѕЇ)
*   **рЅаріарѕЏрѕГріЏ рІерѕџрІѕрІФрІГ AI**: ­ЪЌБ№ИЈ ріе Google Gemini AI рїІрѕГ рЅаріарѕЏрѕГріЏ рѕўрІѕрІФрІерЅх рІГрЅйрѕІрѕЅрЇб AI рІерЅ░рїарЅЃрѕџрІЇріЋ рѕхрѕюрЅх ріерїЇрѕЮрЅх рЅарѕЏрѕхрїѕрЅБрЅх рѕЮрѕІрѕй рѕѕрѕўрѕхрїарЅх рІГрѕъріГрѕФрѕЇрЇб
*   **рѕѕрЅ░рїарЅЃрѕџ рѕЮрЅ╣ рїѕрїйрЅ│**: ­ЪДЉРђЇ­Ъњ╗ рЅа Streamlit ріарѕЏріФріЮріљрЅх рІерЅ░рѕ░рѕФ рЅђрѕІрѕЇріЊ рѕѕріарїарЅЃрЅђрѕЮ рѕЮрЅ╣ рІерѕєріљ рЅарІГріљрїѕрїй ріарѕѕрІЇрЇб
*   **рѕхрѕюрЅхріЋ рѕўрѕ░рѕерЅх рІФрІ░рѕерїѕ рІЇрІГрІГрЅх**: ­Ъўб рЅ░рїарЅЃрѕџрІЇ рІерѕђрІўріЋ рѕхрѕюрЅх ріФрѕ│рІе AI рЅарѕФрѕ▒ рЅ░ріљрѕ│рѕйріљрЅх рІЇрІГрІГрЅх рѕірїђрѕЮрѕГ рІГрЅйрѕІрѕЇрЇб
*   **рЅаріарѕЏрѕГріЏ рІерЅ░рІўрїІрїђ**: ­ЪЄф­ЪЄ╣ рѕЂрѕЅрѕЮ рѕўрѕўрѕфрІФрІјрЅйріЊ рѕўрѕЇрІЋріГрЅХрЅй рЅаріарѕЏрѕГріЏ рЅђрѕГрЅарІІрѕЇрЇб

## РџЎ№ИЈ рІерѕџрІФрѕхрЇѕрѕЇрїЅ ріљрїѕрѕ«рЅй

*   Python (3.8+ рІГрѕўріерѕФрѕЇ)
*   рІері«рѕЮрЇњрІЇрЅ░рѕГ ріФрѕюрѕФ
*   рІерЅарІГріљрѕўрѕерЅЦ рїЇріЋріЎріљрЅх (рѕѕ API ріарїѕрѕЇрїЇрѕјрЅХрЅй)

## ­ЪћД ріарІўрїѕрїЃрїђрЅх (Installation)

1.  **рЇЋрѕ«рїђріГрЅ▒ріЋ рі«рЇњ рІФрІхрѕГрїЅ (Clone the repository):**
    ```bash
    git clone <рІерЇЋрѕ«рїђріГрЅ▒_рѕЏріерѕЏрЅ╗_ріарІхрѕФрѕ╗>
    cd <рІерЇЋрѕ«рїђріГрЅ▒_рѕхрѕЮ>
    ```

2.  **рЅерѕГрЅ╣рІІрѕЇ рібріЋрЅФрІГрѕ«рѕўріЋрЅх (Virtual Environment) рІГрЇЇрїарѕЕріЊ рІФрїЇрЅЦрѕЕ (рІерѕџрѕўріерѕГ):**
    ```bash
    python -m venv venv
    # рЅа Windows рѕІрІГ:
    venv\Scripts\activate
    # рЅа MacOS/Linux рѕІрІГ:
    source venv/bin/activate
    ```

3.  **рІерѕџрІФрѕхрЇѕрѕЇрїЅ рЇЊрігрїєрЅйріЋ рІГрїФріЉ (Install dependencies):**
    рІерѕџріерЅ░рѕЅрЅхріЋ рЇЊрігрїєрЅй рІерІФрІў `requirements.txt` рЇІрІГрѕЇ рІГрЇЇрїарѕЕрЇд
    ```
    streamlit
    opencv-python
    numpy
    google-generativeai
    inference-sdk
    ```
    ріерІџрІФрѕЮ рІГрїФріЉрЇд
    ```bash
    pip install -r requirements.txt
    ```

4.  **рІе Haar Cascade рЇІрІГрѕЇ рѕўріќрѕЕріЋ рІФрѕерїІрїЇрїА:**
    рі«рІ▒ `cv2.data.haarcascades + 'haarcascade_frontalface_default.xml'` рІерѕџрѕѕрІЇріЋ рІГрїарЅђрѕЏрѕЇрЇб `opencv-python` рѕ▓рїФріЋ рІГрѕЁ рЇІрІГрѕЇ ріарЅЦрѕ« рѕўріФрЅ░рЅх ріарѕѕрЅарЅхрЇб ріФрѕЇрЅ░ріФрЅ░рЅ░ рІѕрІГрѕЮ рѕїрѕІ рЅдрЅ│ рѕІрІГ ріерѕєріљ рѕўріЋрїѕрІ▒ріЋ рЅарі«рІ▒ рІЇрѕхрїЦ рѕЏрѕхрЅ░ріФріерѕЇ рІФрѕхрЇѕрѕЇрїІрѕЇрЇб

## ­ЪћЉ рІеріцрЇњріарІГ рЅЂрѕЇрЇјрЅй (API Keys)

рІГрѕЁ рѕўрЅ░рїЇрЅарѕфрІФ рЅарЅхріГріГрѕЇ ріЦріЋрІ▓рѕ░рѕФ рІерѕџріерЅ░рѕЅрЅх рІеріцрЇњріарІГ рЅЂрѕЇрЇјрЅй рІФрѕхрЇѕрѕЇрїЅрЅ│рѕЇрЇд

*   **Google Gemini API Key**: рѕѕ AI рІерІЇрІГрІГрЅх рЅ░рїЇрЅБрѕГрЇб
*   **Roboflow API Key**: рѕѕрѕхрѕюрЅх рѕўрѕѕрІерЅх рѕърІ┤рѕЇрЇб
*   **Roboflow Model ID**: рІеріЦрѕГрѕхрІј рІе Roboflow рЇЋрѕ«рїђріГрЅх рѕърІ┤рѕЇ рѕўрѕѕрІФ (рѕѕрѕЮрѕ│рѕї: `your-project/1`)рЇб
*   **Roboflow API URL**: (ріарЅЦрІЏріЏрІЇ рїірІю `https://detect.roboflow.com` ріљрІЇ)

ріЦріљрІџрѕЁріЋ рЅЂрѕЇрЇјрЅй рЅа Streamlit Secrets рІЇрѕхрїЦ рѕЏрѕхрЅђрѕўрїЦ рІФрѕхрЇѕрѕЇрїІрѕЇрЇб рЅарЇЋрѕ«рїђріГрЅхрІј рІІріЊ рІ│рІГрѕгріГрЅХрѕф рІЇрѕхрїЦ `.streamlit` рІерѕџрЅБрѕЇ рЇјрѕЇрІ░рѕГ рІГрЇЇрїарѕЕріЊ рЅарІЇрѕхрїА `secrets.toml` рІерѕџрЅБрѕЇ рЇІрІГрѕЇ рІГрЇЇрїарѕЕрЇб рІерЇІрІГрѕЅ рІГрІўрЅх ріЦріЋрІ▓рѕЁ рѕўрѕЮрѕ░рѕЇ ріарѕѕрЅарЅхрЇд

```toml
GEMINI_API_KEY = "рІеріЦрѕГрѕхрІј_GEMINI_API_KEY_ріЦрІџрѕЁ_рІФрѕхрїѕрЅА"
ROBOFLOW_API_KEY = "рІеріЦрѕГрѕхрІј_ROBOFLOW_API_KEY_ріЦрІџрѕЁ_рІФрѕхрїѕрЅА"
ROBOFLOW_MODEL_ID = "рІеріЦрѕГрѕхрІј_ROBOFLOW_MODEL_ID_ріЦрІџрѕЁ_рІФрѕхрїѕрЅА" # рѕЮрѕ│рѕї: "emotion-detection-model/3"
ROBOFLOW_API_URL = "https://detect.roboflow.com"
```

рѕЏрѕ│рѕ░рЅбрІФ: "рІеріЦрѕГрѕхрІј_..." рІерѕџрѕЅрЅхріЋ рЅдрЅ│рІјрЅй рЅарѕФрѕхрІј рЅхріГріГрѕѕріЏ рЅЂрѕЇрЇјрЅй рІГрЅ░ріЕрЇб

РќХ№ИЈ рѕўрЅ░рїЇрЅарѕфрІФрІЇріЋ рѕЏрѕхрігрІх (Running the Application)

рѕЂрѕЅрѕЮ рІЮрїЇрїЁрЅХрЅй ріерЅ░рїаріЊрЅђрЅЂ рЅаріІрѕІ рѕўрЅ░рїЇрЅарѕфрІФрІЇріЋ рѕѕрѕЏрѕхрігрІхрЇд
```bash
streamlit run main.py
```



­Ъџђ ріарїарЅЃрЅђрѕЮ

рѕўрЅ░рїЇрЅарѕфрІФрІЇ рѕ▓ріерЇѕрЅх рѕхрѕЮрІјріЋ ріЦріЋрІ▓рІФрѕхрїѕрЅА рІГрїарІГрЅЁрІјрЅ│рѕЇрЇб рѕхрѕЮрІјріЋ рІФрѕхрїѕрЅАріЊ "ріарѕхрїѕрЅБ / Submit" рІерѕџрѕѕрІЇріЋ рІГрїФріЉрЇб

рЅарїЇрѕФ рЅаріЕрѕЇ рЅБрѕѕрІЇ рІерѕўрЅєрїБрїарѕфрІФ ріГрЇЇрѕЇ рІЇрѕхрїЦ "­ЪЊи ріФрѕюрѕФ ріарЅЦрѕФ/ріарїЦрЇІ" рІерѕџрѕѕрІЇріЋ рЅЂрѕЇрЇЇ рЅарѕўрїФріЋ ріФрѕюрѕФрІјріЋ рІФрЅЦрѕЕрЇб

ріФрѕюрѕФрІЇ рѕ▓рЅарѕФ рѕўрЅ░рїЇрЅарѕфрІФрІЇ рЇірЅхрІјріЋ рѕѕрѕўрѕѕрІерЅх рІГрѕъріГрѕФрѕЇ ріЦріЊ ріерЅ░рѕ│ріФ рѕхрѕюрЅхрІјріЋ рІФрѕ│рІФрѕЇрЇб

рЅарЅђріЮ рЅаріЕрѕЇ рЅБрѕѕрІЇ рІе AI Chat ріГрЇЇрѕЇ рІЇрѕхрїЦ ріе AI рїІрѕГ рЅаріарѕЏрѕГріЏ рѕўрІѕрІФрІерЅх рІГрЅйрѕІрѕЅрЇб рѕўрѕЇрІЋріГрЅхрІјріЋ рІФрѕхрїѕрЅАріЊ "рѕІріГ" рІерѕџрѕѕрІЇріЋ рІГрїФріЉрЇб

ріФрѕюрѕФрІЇ рІерЅарѕФ ріерѕєріљріЊ рІерѕђрІўріЋ рѕхрѕюрЅх ріерЅ│рІе AI рЅарѕФрѕ▒ рІЇрІГрІГрЅх рѕірїђрѕЮрѕГ рІГрЅйрѕІрѕЇрЇб

­ЪЊю рѕЏрѕ│рѕ░рЅбрІФ

рІерѕхрѕюрЅх рѕўрѕѕрІерЅ▒ рЅхріГріГрѕѕріЏріљрЅх рЅаріФрѕюрѕФрІЇ рїЦрѕФрЅхрЇБ рЅарЅЦрѕГрѕЃріЋ рѕЂріћрЅ│ ріЦріЊ рЅа Roboflow рѕърІ┤рѕЇрІј ріарЅЁрѕЮ рѕІрІГ рІерЅ░рѕўрѕ░рѕерЅ░ ріљрІЇрЇб

рІГрѕЁ рѕўрЅ░рїЇрЅарѕфрІФ рѕѕрѕЎрІФрІі рІерѕЁріГрѕЮріЊ рІѕрІГрѕЮ рІерѕхріљ-рѕЇрЅдріЊ рѕЮріГрѕГ рѕЮрЅхріГ ріарІГрІ░рѕѕрѕЮрЇб ріерЅБрІх рІерѕхрѕюрЅх рЅйрїЇрѕГ ріФрѕѕрЅЦрІј ріЦрЅБріГрІј рЅБрѕѕрѕЎрІФ рІФрѕЏріГрѕЕрЇб

рІГрѕЁріЋ рЇЋрѕ«рїђріГрЅх рѕхрѕѕрЅ░рѕўрѕѕріерЅ▒ ріЦріЊрѕўрѕ░рїЇріЊрѕѕріЋ! рѕЏріЋріЏрІЇрѕЮ ріарѕхрЅ░рІФрІерЅх рІѕрІГрѕЮ рїЦрІФрЅё ріФрѕѕрІјрЅх рѕѕрѕЏрѕ│рІѕрЅЁ ріарІФрѕўріЋрЅ▒рЇб
